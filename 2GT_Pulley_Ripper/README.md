# 2GT Pulley Ripper

## Information
Stock Voron/Monolith BOM does not include M5x50 SHCS. The original model was a shy too long to use M5x40 SHCS, which should already be on hand.

I highly recommend getting the clamping sleeve as it can be more difficult to hold the smaller dehubbing tool.
